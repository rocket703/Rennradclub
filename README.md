# Magdeburger Radsportclu e.V. - Webseite 2026

Dies ist die offizielle Webseite des Rennradclubs Magdeburg. 
Das Projekt wurde mit Fokus auf Speed, Clean-Design und Barrierefreiheit entwickelt.

## Features
- Modernes "Reveal-from-Bottom" Design
- Responsive Layout (Mobile & Desktop optimiert)
- Interaktive Event-Karten mit 3D-Flip-Effekt
- Kontaktformular im r3-Minimalismus-Stil

## Technologien
- HTML5
- CSS3 (Custom Properties / Flexbox / Grid)
- JavaScript (Vanilla JS)
